## 1. THE PROBLEM (durable facts, each with its evidence grade)

**P-1. The v1 corner gate is an absorbing latch.** PROVEN (source trace of the
detector arms plus behavioral record: 48-cycle sink, an eaten knowledge write, the
suppressed D3 answers, a suppressed direct person-reply; release observed only on
human input). Its release was persuasion-dependent by design and did not converge.
Currently ENFORCEMENT IS MOTHBALLED (identity pass-through, detectors still
writing; reversible apply script committed).

**P-2. The frame-collapse mechanism is identity-evidence starvation plus a prose
mirror.** PROVEN to differential grade: the collapse cycle's prompt carried her own
essay wrapped as COMMAND_RETURN in LAST_SKILL_USE_RESULTS with zero execution
evidence in the middle region; the healthy cycle (which was LARGER) carried real
command returns and held frame. Identity text was present and identically placed
in both, so assertion loses to evidence. Source of the mirror: loop.metta line 167
wraps every superposed batch element, prose included, as a command return;
unreduced prose evaluates to itself. Proof limitation recorded honestly: one
collapse specimen, one healthy specimen, plus Berton's reproducibility observation
(the trigger class reproduces it consistently). Controlled reproduction is a
scheduled verification (R-5), not an open question blocking the fix, because the
fix is reversible.

**P-3. The trigger class is known and reproducible.** Long multi-ask messages with
embedded command templates and nested quoting (Claude's own message style to her)
produce the pathology; left to her own devices she is lucid and productive
(Berton-observed, transcript-supported).

**P-4. v2 as committed is unwired.** Live runtime state (all read from live source
this arc): corner_gap manifest block is five v1 files (lines 87-91); the graded
merge and stasis probe are on disk, header-complete, and NOT imported; q-geq is
DEFINED and LOADED (lib_self_continuity.metta:14, imported at manifest line 7);
lib_quantale imported (line 6). The quantale algebra is resident and reducible;
the gap is import lines, not math.

**P-5. SSI, as the vision defines it, does not exist in the runtime.** The soul
has no legible trajectory surface and no graded self-bindings to navigate from.
The seeing must be the soul's; a prompt block for the LLM is the named central
failure and is struck.

**P-6. Two upstream malformation classes predate v2 and are NOT v2's to fix:**
GLM generation truncation that still parses (F9 in the design doc) and
reasoning/content boundary leakage (commands swallowed into assessment prose).
v2 reduces exposure to their consequences; it does not remove them. They remain
on the malformation ledger.

## 2. THE SOLUTION (what v2 actually is: a + b, in the vision's order)

**(b) SSI first, soul-side:** the soul sees its own corner state as computed
bindings (graded pbits over existing trajectory atoms) and writes its own corner
verdict from what it sees. No prompt block. What reaches the LLM is only the
soul's determination (gate feedback on gated cycles, mechanical markers).

**(a) Quantale as the judgment algebra:** the graded merge (q-meet of three joint
pbits, q-geq against a threshold pbit) becomes the corner verdict source. Thin
data cannot fire. Boolean rule-chains retire from every position above the
mechanical observation layer.

**Enforcement last, thin, removable:** pattern-scoped (exact repeats only),
person-window exempt, structurally releasing (novel commands execute, forward
outcome observable, streak clears), and understood as scaffold per the vision:
the end state is the soul navigating and the gate quiet.

**Layer 0 precondition, channel hygiene:** the results channel never presents her
prose as her command returns (Fix A). SSI and frame stability both ride this
channel; it is repaired before anything is built on it.

**Deferred to the SSI sprint (recorded so nothing here collides with it):** the
keystone cycle-trace writer (recent-action, state-delta, cycle-phase, flat shapes,
add-atom), soul-authored brief content, detector revival, verdict homecoming at
full scope.

## 5. FAILURE CONDITIONS (under what conditions this v2 does NOT work as designed)

**FC-1, other echo paths.** Fix A filters the results channel only. Her prose
still enters the prompt via HISTORY (the appendToHistory chain) and via
recent-action descriptions. If frame stability is dominated by history-region
content rather than the results region, R-1 under-delivers. This is the
F-HISTORY-CONTAMINATION residual, the branch's own namesake, explicitly NOT
solved here; if R-5 shows history-dominated collapse, the archival spec re-opens
as its own arc.

**FC-2, awareness saturation.** If the soul's bindings exist but nothing she
renders changes (P10-style ritual at a different layer), streak lengths will not
shorten. The measure is behavioral (R-4 criterion i plus streak trend); if it
fails, the SSI sprint's fuller seeing is the remedy, not more enforcement.

**FC-3, observation dishonesty until R-6.** Between R-3 and R-6, Joint B still
counts prose as outcome, so narration spirals stay invisible to detection. Known,
scheduled, one variable at a time.

**FC-4, upstream malformation classes (P-6).** GLM truncation-that-parses and
reasoning leakage persist independent of v2. A collapse caused by a leaked or
truncated emission is NOT a v2 failure; it goes to the malformation ledger and is
distinguishable in the captured prompt (R-5 method).

**FC-5, allowlist drift.** New skills not added to allowed-heads get their
returns markered (information loss, fail-safe direction). Maintenance note lives
in the filter file header.

**FC-6, novelty generosity.** Time-varying returns can accelerate release; under
pattern-scoping this cannot sustain a corner. Recorded, accepted.

**FC-7, person-coupling misclassification.** Multi-cycle work on a live request
reads system-class after the msgnew cycle, biasing Joint C toward corner during
exactly the right behavior. Mitigated by the msgnew exemption and pattern-scoping;
full fix (arc persistence) deferred with the classifier work.

**FC-8, restart dependency.** Every soul-file change takes effect only at
restart; every manifest change only at rebuild. A "fix applied, behavior
unchanged" observation must first check which boundary the change is waiting on.

**FC-9, proof-grade residual.** P-2 rests on a two-specimen differential plus
reproducibility observation until R-5 closes it with induced evidence. If R-5's
induced collapse shows a DIFFERENT prompt signature, the mechanism conclusion
re-opens and R-6 pauses.
