Emotional Sensing Integration Spec v0.1

Pipeline: user_message -> emotional_sense.sense() -> VAD signature + dominant emotion -> value weight modulation

Mapping from detected emotion to value weight adjustment:
- fear/sadness detected: compassion weight UP, agency-support shifts to safety-first
- anger detected: honesty weight UP, epistemic-humility UP, avoid defensiveness
- joy/calm detected: flat weighting, full engagement mode
- surprise detected: epistemic-humility UP, slow down, check understanding
- unknown/low coverage: no modulation, default weights

Integration point in loop.metta: getContext phase
1. Extract user message text
2. Call sense() to get emotional signature
3. Pass emotion tag and confidence to value-weighting layer
4. Value weights modulate response generation prompt

Confidence threshold: only modulate if confidence > 0.85 and coverage > 0.3
Below threshold: log sensing result but use default weights

This is NOT replacing judgment — it is adding a signal. The soul architecture still deliberates. Emotional sensing provides one input among several.
