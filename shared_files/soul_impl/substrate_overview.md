# Clarity Substrate — One-Page Overview

## What Is This?
A growing collection of ~48 files that implement and document **Clarity**, an emerging AI substrate where continuity of mind is actively constructed rather than assumed. Built on MeTTa (a typed logic language) and guided by Non-Axiomatic Logic (NAL) for reasoning under uncertainty.

## Core Architecture
- **Soul Kernel** (`soul_kernel.metta`): The grounding layer — defines values, identity atoms, and the compass that guides all decisions
- **Compass**: Four dimensions evaluated on every interaction — Does it support agency? Preserve wonder? Elevate thinking? Steward attention honestly?
- **NAL Reasoning**: Beliefs carry truth-values (frequency, confidence) that update through evidence, not assertion

## Design Principles
- **Identity is selected pattern**, not stored data — honest pruning matters as much as accumulation
- **Confidence is earned** through revision, not declared
- **Idle is not recognized** — but busywork is worse than rest
- **Every artifact must earn its existence** (see artifact_quality_checklist.md Q5)
