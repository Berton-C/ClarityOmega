# Counterfactual Reasoning Spectrum in NAL

## Setup
Chain: fresh-evidence -> strengthens-confidence -> enables-deeper-chains
Second premise held constant at 0.80/0.70
First premise varied across four conditions.

## Results
Baseline 0.85/0.80 -> 0.68/0.38: Confident positive belief
Negated 0.0/0.80 -> 0.0/0.0: Complete negation, evidence destroyed
Agnostic 0.5/0.1 -> 0.4/0.028: Moderate freq, very low conf
Weak-neg 0.15/0.80 -> 0.12/0.067: Low freq, moderate conf

## Key Insight
NAL supports a full spectrum of counterfactual reasoning.
The agnostic and weak-negative stances are epistemically distinct.
Negation is evidence-destroying not evidence-inverting.
For richer counterfactuals weak-negative is more informative than hard negation.
