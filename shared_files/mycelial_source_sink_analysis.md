# Source-Sink Transport Logic: Fourth Mechanism?
Date: 2025-04-15

## The Question
Do mycelial source-sink dynamics map onto a memory mechanism BEYOND the three already proposed?

## Three Existing Mechanisms
1. Retrieval-weight reinforcement (passive: high-weight memories score higher)
2. Anastomosis linking (structural: co-retrieved memories form connections)
3. Quorum synthesis (emergent: dense clusters generate abstractions)

## Source-Sink Transport in Mycelium
- Nutrients flow bidirectionally along concentration gradients
- Sources = nutrient-rich nodes; Sinks = nutrient-depleted nodes
- Flow is PROPORTIONAL to gradient magnitude
- Sources and sinks are dynamic — a source becomes a sink when depleted
- Key: the network actively reallocates, not just passively available

## Proposed Fourth Mechanism: Demand-Driven Retrieval Push
- Query = sink (information-depleted context)
- High-weight memories = sources (information-rich nodes)
- Current system: query pulls from memory (sink-driven only)
- Proposed: high-weight memories could PUSH into contexts where they are partially relevant
- This is associative priming — memories above a weight threshold activate nearby memories even without direct query match
- Biological analog: nutrient-rich hyphae push resources toward connected depleted regions WITHOUT the depleted region signaling

## Testable Difference
- Without mechanism 4: only query-similar memories retrieved
- With mechanism 4: high-weight memories activate their anastomosis-linked neighbors, which may surface in retrievals they would not otherwise appear in
- This creates SPREADING ACTIVATION through the weight-link topology

## Assessment
This IS genuinely distinct from mechanisms 1-3. It turns the memory network from pull-only to push-pull. The biological grounding is strong — bidirectional transport is confirmed. The computational analog is spreading activation, which has deep roots in cognitive science (Collins & Loftus 1975).

## Risk
Unbounded spreading activation = noise. Need dampening: activation decays with link distance, only fires above weight threshold.
