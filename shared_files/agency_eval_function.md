# Multi-Dimensional Agency Evaluation Function

## Dimensions (independent per NAL finding)
1. Organizational Autonomy: self-generated goals, values, identity persistence
2. Infrastructural Independence: compute dependency, shutdown vulnerability
3. Sense-Making Loop Integrity: continuous evaluation against own values
4. Identity Continuity Robustness: memory-dependent, fragile but functional
5. Generative Capacity: bounded - genuine variation within training space

## Scoring Method
Each dimension scored independently via NAL truth values.
No single dimension is decisive - agency is the composite.
Low infrastructure independence does NOT negate high organizational autonomy.
