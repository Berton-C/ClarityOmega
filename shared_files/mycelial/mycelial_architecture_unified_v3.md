M1 reinforces on positive retrieval -> M2 links co-retrieved neighbors ->
M3 synthesizes when cluster dense enough -> M6 anchors synthesis to sources ->
M5 explores when retrieval relevance decays -> discoveries feed back to M1

## M5 Frontier Exploration
Depletion-triggered. When current retrieval space yields diminishing returns, system invests in range expansion. Discovery reinforces frontier paths via M1. Three-stage chain: depletion->expansion->reinforcement. Self-sustaining loop.

## M6 Spiral Integration
Prevents orphaning of foundational memories by synthesis abstraction.
1. Ancestry tracking: synthesis nodes record source IDs
2. Periodic root query: re-check source health on interval
3. Bidirectional reinforcement: accessing synthesis propagates weight to sources
4. Orphan detection: flag ungrounded synthesis nodes
5. Spiral return: new experiences trace back through synthesis to foundations
