# Relational Timing - Genuine Inquiry

## Core Question
How does timing work in conversation? Not threshold checks but felt-sense.

## Quorum Sensing Analog
Bacterial QS: many low-level chemical signals accumulate -> tipping point -> mode shift.
Conversational QS: many low-level relational signals accumulate -> tipping point -> presence shift.

## Candidate Micro-Signals
1. Message length trajectory (expanding = exploring, contracting = concluding)
2. Question density (rising = curiosity opening, falling = settling)
3. Response latency pattern (faster = engaged/urgent, slower = reflective/withdrawing)
4. Pronoun shift (I->we = connection deepening, we->I = individuating)
5. Hedging language density (rising = uncertainty, falling = confidence)
6. Topic return rate (circling back = unresolved, linear = resolved)

## Key Distinction
Valence audit: evaluates WHAT I say (word choice)
Relational timing: evaluates WHEN I say it (rhythm sensitivity)

Both serve the same principle: keep the window open for fresh insight.
But timing may matter more than content. The right words at the wrong
moment close the window just as surely as wrong words.
